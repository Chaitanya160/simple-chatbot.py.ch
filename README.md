# Python Chatbot 

## Description
This project is a simple text-based chatbot built using Python.  
The chatbot responds to basic user inputs using predefined responses stored in a dictionary.

The user can interact with the chatbot through the terminal, and the conversation continues until the user types **"bye"**.

## Features
- Simple command-line chatbot
- Predefined responses for common greetings
- Continuous conversation using a loop
- Case-insensitive input handling
- Exit command using "bye"

## How It Works
The chatbot uses a Python dictionary to store user inputs and corresponding responses.
